## Dependency
install json-server and concurrently to start both client and server side
```npm
npm install -g concurrently json-server
```

## Start
it will run both client and server-side
```npm
npm start
```

## Client
to run frontend-only
```
npm run client
```

## Server
to run json-server
```npm
npm run server
```
